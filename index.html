001

aggeliki.dev — Cloud Portfolio

Live

Personal portfolio deployed on AWS using a fully serverless architecture. The frontend is hosted on S3 and distributed globally via CloudFront, with a custom domain (www.aggeliki.dev), HTTPS via SSL/TLS, and edge caching for low-latency delivery worldwide. Dynamic functionality is handled by a serverless visitor counter built with AWS Lambda, API Gateway, and DynamoDB — real-time data within a static site. Demonstrates end-to-end cloud deployment: DNS resolution, CDN configuration, serverless compute, and secure web hosting.
