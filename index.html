Development & Deployment Workflow

This project demonstrates a simple and practical static website workflow using Git, GitHub, and GitHub Pages. Every update to the website follows a structured development process:

Plan → Build in VS Code → Git Commit → Push to GitHub → Deploy with GitHub Pages

From an architectural perspective, the website is delivered as a static web application. Users access the site through GitHub Pages hosting, which serves the HTML, CSS, and JavaScript files publicly on the web.

User → Internet → GitHub Pages → Static Portfolio Website
